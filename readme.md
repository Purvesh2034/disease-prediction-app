# 💓 AI-Based Disease Risk Prediction System

## 📌 Project Overview

This project is a Machine Learning-based web application that predicts the risk of heart disease using patient health parameters. The system uses a Random Forest classifier and is deployed using Streamlit for real-time interaction.

---

## 🎯 Objectives

* Analyze healthcare data using EDA
* Build predictive ML models
* Compare multiple algorithms
* Deploy an interactive web application

---

## 🧠 Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit

---

## 📊 Machine Learning Models

* Random Forest Classifier (Final Model)
* Logistic Regression (Comparison Model)

---

## 📈 Results

* Random Forest Accuracy: ~88%
* Logistic Regression Accuracy: ~82%
